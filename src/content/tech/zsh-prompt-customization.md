## はじめに

ターミナルを画面共有するとき、デフォルトのプロンプトには個人情報が丸見えになっています。

```text
Hoge@HogenoMacBook-Air ~ %
```

うまく隠せないかなと思い色々試してみました。その際に実施したことを整理したいと思います。なお、今回はmacOS（zsh）で実施しています。


## プロンプトの仕組み

zsh では `PROMPT` という変数にプロンプトの見た目が定義されています。この変数を書き換えるだけで、見た目を自由に変えられます。bash の場合は `PROMPT` の代わりに `PS1` を使います。

よく使う記号は以下の通りです。

| 記号 | 意味 |
|------|------|
| `%~` | カレントディレクトリ（`~` 省略あり） |
| `%/` | カレントディレクトリ（絶対パス） |
| `%1~` | カレントディレクトリ名だけ |
| `%n` | ユーザー名 |
| `%m` | ホスト名（マシン名） |
| `%F{色}〜%f` | 色付き表示 |

https://zsh.sourceforge.io/Doc/Release/Prompt-Expansion.html

## 方法1：その場で一時変更（セッション限り）

```zsh
PROMPT="%~ $ "
```

ターミナルを閉じれば元に戻ります。発表のときだけ使いたいなら、これで十分です。

ターミナルを閉じずに元のプロンプトに戻したい場合は、以下を実行するとシェルが再起動されてデフォルトに戻ります。

```zsh
exec zsh
```

## 方法2：関数にして .zshrc に登録しておく

毎回打つのが面倒なら、`~/.zshrc` に関数を仕込んでおく方法があります。

`~/.zshrc` に以下を追記します。

```zsh
present() {
  PROMPT="%~ $ "
}
```

設定を反映します。

```zsh
source ~/.zshrc
```

以降は `present` と打つだけで切り替わります。


## カスタマイズ例

```zsh
# シンプル
PROMPT="%~ $ "

# 色付き（ディレクトリを緑に）
PROMPT="%F{green}%~%f $ "

# 矢印スタイル
PROMPT="%~ ❯ "

# ディレクトリ名だけ
PROMPT="%1~ $ "
```


## おわりに

プロンプトは `PROMPT` 変数を書き換えるだけで変えられます。一時的に変えたいだけなら `PROMPT="..."` をその場で実行すれば十分ですし、繰り返し使うなら `.zshrc` に関数として登録しておくと便利です。
